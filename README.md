# 🌍 Travel Website Assignment

## 🎯 Objective

Build a responsive Travel & Tour website using HTML and CSS.

Your submission will be **automatically evaluated using GitHub Actions**.

---

## 🧩 Requirements

You MUST implement the following sections:

### 1. Navbar

* Class: `.navbar`
* Should contain logo and navigation links

### 2. Hero Section

* Class: `.hero`
* Must include heading and short description

### 3. Destinations Section

* Class: `.destinations`
* Must contain at least **3 destination cards**

Each `.destination-card` must include:

* Image → `.destination-img`
* Title → `.destination-title`
* Price → `.destination-price`

### 4. Booking Button

* Class: `.book-btn`
* Should be visible and styled

### 5. Footer

* Class: `.footer`

---

## ⚠️ Important Rules

* You MUST use the exact class names given above
* Do NOT rename classes (tests depend on them)
* All images must have `alt` attribute
* Minimum 3 destination cards required

---

## 🧪 Evaluation Criteria

Your project will be tested automatically for:

* ✅ Required sections exist
* ✅ Correct class names used
* ✅ Minimum number of destination cards
* ✅ Each card contains title and price
* ✅ Button and footer present

---

## 🚀 How to Submit

Follow these steps:

### Step 1: Clone the repository

```bash
git clone <your-repo-link>
cd travel-template
```

### Step 2: Make your changes

* Edit `index.html`
* Add your styles in `styles.css`

### Step 3: Install dependencies

```bash
npm install
```

### Step 4: Test locally (optional)

```bash
npm test
```

### Step 5: Commit and push

```bash
git add .
git commit -m "Completed travel website"
git push
```

---

## ⚡ Automatic Testing

Once you push your code:

* GitHub Actions will run automatically
* Your code will be tested
* You can check results in the **Actions tab**

---

## 📊 Result

* 🟢 All tests pass → Assignment successful
* 🔴 Any test fails → Fix your code and push again

---

## 💡 Tip

If your tests fail:

* Check class names carefully
* Ensure all required elements exist

---

Good luck 🚀
